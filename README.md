# CloudTok

CloudTok is a full-stack short-video social media platform built from scratch to explore modern web development, cloud architecture, and real-world software engineering practices.

The project focuses on creating a platform where users can discover, watch, upload, and interact with short-form video content while learning how professional applications are designed, developed, tested, and deployed.

## Project Goals

CloudTok aims to explore:

- Frontend development
- Backend API development
- Database design
- Cloud storage systems
- User authentication
- Social media features
- Real-world deployment workflows

## Current Architecture

## Development Philosophy

CloudTok is developed using an iterative process:
Build → Test → Debug → Improve → Document


Each feature is developed, tested, and improved before moving to the next stage.
